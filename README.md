## What is this?

Pillar OS is a compliance management platform for UK supported housing, children's homes, care homes, and domiciliary care providers.

This demo showcases the full system with realistic data — built to show commissioners, regulators, and care providers what modern compliance infrastructure looks like.

## Features

| Module | Description |
|--------|-------------|
| **Compliance Dashboard** | Real-time overview of organisational compliance status |
| **Support Worker Dashboard** | Daily tasks, SU list, contact events, pending actions |
| **Service User Profile** | Complete SU view — documents, contacts, incidents, risk |
| **Incidents & Risk** | Incident management, CfC, abandonment, Safe & Well tracking |
| **Reports & QA** | Weekly compliance reports, QA scoring, team performance |
| **Contact Notes + AI QA** | Core daily workflow with AI quality review |
| **Lone Working** | Real-time staff safety monitoring with check-in/out |
| **Properties & Voids** | Property management, void tracking, move-in pipeline |
| **Team Leader Dashboard** | Team oversight, verification queue, escalations |
| **Support Plans** | SP tracking, review scheduling, goal progress |
| **SU Journey** | Referral-to-active intake pipeline |
| **SAO Compliance** | Staff compliance status overview |
| **Document Management** | Cross-SU document tracking and gap analysis |
| **Access Control** | Role-based permissions and GDPR compliance |
